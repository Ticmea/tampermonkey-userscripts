# "Mein Mix" starten

"Mein Mix" starten ist ein Userscript, dass es dem Benutzer erlauben soll YouTubes "Mein Mix" und "Mix - [Video Titel]" Playlisten manuell zu starten.

Es registriert zwei Menüeinträge, welche den Benutzer zur "Mein Mix" Playlist weiterleiten, wenn sie angeklickt werden.
(Der Eine startet vom aktuellen Video aus sofern verfügbar, der Andere startet von einem 0-Sekunden Video aus)

Es registriert außerdem einen Menüeintrag, welcher den Benutzer (vom aktuellen Video aus) zur "Mix - [Video Titel]" Playlist weiterleitet, wenn es angeklickt wird.

Die zwei Menüeinträge, welche den Benutzer vom aktuellen Video zu einer Liste weiterleiten, werden nur dann registriert, wenn zum Zeitpunkt zu dem das Skript ausgeführt wird, ein Video ausgewählt ist.

Sie finden das Skript installationsbereit unter folgendem Link: https://greasyfork.org/de/scripts/565457-youtube-start-my-mix
