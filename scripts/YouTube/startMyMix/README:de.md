# "Mein Mix" starten

"Mein Mix" starten ist ein Userscript, dass es dem Benutzer erlauben soll YouTubes "Mein Mix" Playlist manuell zu starten.

Zu diesem zweck registriert es zwei Menüeinträge, welche den Benutzer zu dieser Playlist weiterleiten, wenn sie angeklickt werden.
(Das Eine startet vom aktuellen Video aus sofern verfügbar, das Andere started von einem 0-Sekunden Video aus)

Sie finden das Skript installationsbereit unter folgendem Link: https://greasyfork.org/de/scripts/565457-youtube-start-my-mix
